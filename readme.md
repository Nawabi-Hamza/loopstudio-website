# Loopstudios Website

Loopstudios landing page Tailwind 

## Usage
Clone Repo
```bash
git clone https://github.com/Nawabi-Hamza/loopstudio-website.git
```

Install dependencies

```bash
npm Install
```

Run Tailwind CLI

```bash
npm run watch
```
